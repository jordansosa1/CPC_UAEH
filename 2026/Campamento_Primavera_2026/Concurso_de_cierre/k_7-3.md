# K. 7 - 3

## Descripción

*"Me llama y me dice..."*

Coco y Felipe discuten sobre la anatomía de los perros. Parece ser que hasta este punto de la conversación solo quieren contradecirse el uno al otro, así que ayúdalos a resolver una cuestión tan simple, ¿cuántas patas tiene un perro?

## Entrada
Este problema no tiene entrada.

## Salida
Imprime cuántas patas tiene un perro.

## Ejemplos

### Salida
```
4
```

## Temas identificados
### Programación
- Console Output

## Propuesta de solución
Solo hay que imprimir $4$.

## Implementación
Imprimir $4$.

### C++

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    cout << "4";

    return 0;
}
```
